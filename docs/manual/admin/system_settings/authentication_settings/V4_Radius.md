# Radius 认证

## 关于 RADIUS

!!! tip "RADIUS"
    - **远程身份认证拨入用户服务（RADIUS）** 是一种基于 RADIUS 协议的远程身份认证机制，主要用于网络访问控制。它提供身份认证、授权和计费（AAA）功能。

## 如何配置

!!! tip ""
    - 在页面右上角，单击设置。
    - 导航到 **系统设置 > 认证设置 > Slack。**
    - 在 **Radius** 字段中，选中以启用 Radius 身份认证。
    - 在 **主机** 字段中，填入 RADIUS 服务器IP地址或域名，例如“172.16.10.180”。
    - 在 **端口** 字段中，填入 RADIUS 服务器端口号。默认值为“1812”。
    - 在 **密文** 字段中，填入 JumpServer 和 RADIUS 服务器之间的共享密钥。它的功能类似于密码，对 RADIUS 请求和响应中的敏感信息进行加密，以确保安全通信。
    - 可选地，在 **RADIUS 中的 OTP** 字段中，选中以启用 RADIUS 作为 MFA 后端。有关更多信息，请参阅启用 RADIUS MFA 后端。
    - 在 **组织** 字段中，经过身份认证和创建后，用户将被添加到所选组织中。
    - 点击 **提交**。

启用 RADIUS MFA 后端

!!! tip "本小节介绍如何将 RADIUS 身份认证配置为 MFA 后端之一。"
    - 按照集成 RADIUS 身份认证指南配置 RADIUS 身份认证。
    - 在 **RADIUS 中的 OTP** 字段中，选中启用 RADIUS 作为 MFA 后端。当用户的 MFA 启用时，他们可以在登录时选择 RADIUS 身份认证类型。
    - 点击 **提交**。

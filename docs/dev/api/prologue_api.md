# 前言

## 概述

感谢您选择JumpServer 堡垒机。本手册详细描述了JumpServer 堡垒机（以下简称“JumpServer”或“系统”）的部分常见 API 使用方法。

本文档的 API 请求头中的 X-JMS-ORG 参数为企业版的组织管理功能，留空则默认为 Default 组织。

因版本升级或不同项目配置文件不相同等原因，手册中所提供的 API 内容与用户使用的可能不一致，请以当前版本在线 Swagger UI 中的 API 信息为准，手册中不再针对前述情况造成的差异一一说明。

出于功能介绍及配置示例的需要，手册中可能会使用IP地址、网址、域名等。如无特殊说明上述内容均为示意，不指代任何实际意义。

## 预期读者

本文档主要适用于使用 JumpServer 的开发人员，包括系统对接、二次开发人员等。读者应具备以下基础知识：

- REST API 的基本使用方法
- 编程语言基础（如 Python、Bash 等）
- 堡垒机系统的基本使用方法与核心产品概念
